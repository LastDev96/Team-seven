x. Điều khiển dạng nút

Nút dạng button có tác dụng bắt sự kiện click (thường được sử dụng kết hợp vs JS)

Loại submit

submit có tác dụng đẩy toàn bộ dữ liệu trên form về server để xử lý

1 form thường chỉ có 1 nút submit

Nút reset có tác dụng khởi tạo lại các đối tượng trên form về giá trị mặc định

Đối tượng file

Cho phép người dùng duyệt thư mục và chọn các file trên máy tính client, và lưu lại địa chỉ vật lý của file đó phục vụ upload lên server
